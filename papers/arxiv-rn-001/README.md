# arXiv Submission — RN-001: The Continuity Problem

## How to Submit

1. Go to https://arxiv.org → START NEW SUBMISSION
2. Upload `paper.tex` as the main file
3. Select archive: **cs.CR** (Cryptography and Security)
4. Title auto-populates from the `\title{}` field
5. Authors: **The Continuity Lab**
6. Abstract auto-populates from paper
7. Add `https://www.myshape.com/research` as report number
8. Submit

## Notes

- No figures, no external dependencies — pure LaTeX, arxiv compiles directly
- Uses IEEEtran.cls (standard, available on all arxiv build systems)
- arXiv will auto-assign an ID like `arXiv:2607.XXXXX`
- Once published, add the arXiv ID back to the website as a citation badge
